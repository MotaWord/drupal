-----------------------
- Changelog, readme updates.


TMGMT - MotaWord Translator 11.0.0, 2021-04-15
-----------------------
This is a major update to support more complex workflows and larger websites.

- The module now provides an option to process each post/page as a separate source file at MotaWord
- MotaWord translation projects with multiple source files are delivered to Drupal as a ZIP file. The module now handles ZIP package delivery.
